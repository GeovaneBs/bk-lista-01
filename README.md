# Lista de Exercùcios de Lùgica de Programaùùo em C#

## Entrada, Processamento e Saùda de Dados

1. **Calcular o volume de uma caixa retangular.**
   - Exemplo de entrada: Altura = 5, Largura = 3, Comprimento = 2
   - Fùrmula: Volume = Altura * Largura * Comprimento
   - Exemplo de saùda: O volume da caixa ù 30 unidades cùbicas.

2. **Converter a temperatura de graus Fahrenheit para graus Celsius.**
   - Exemplo de entrada: Temperatura em Fahrenheit = 68
   - Fùrmula: Celsius = (Fahrenheit - 32) * 5/9
   - Exemplo de saùda: A temperatura em Celsius ù 20 ùC.

3. **Calcular o consumo de combustùvel por quilùmetro.**
   - Exemplo de entrada: Quilometragem inicial = 100 km, Quilometragem final = 200 km, Consumo de combustùvel = 20 litros
   - Fùrmula: Consumo por km = Consumo de combustùvel / (Quilometragem final - Quilometragem inicial)
   - Exemplo de saùda: O consumo de combustùvel por quilùmetro ù de 0.1 litros/km.

4. **Calcular o salùrio total de um vendedor.**
   - Exemplo de entrada: Salùrio base = R$ 1500, Total de vendas = R$ 5000, Comissùo (5% sobre as vendas)
   - Fùrmula: Salùrio total = Salùrio base + (Total de vendas * Comissùo)
   - Exemplo de saùda: O salùrio total do vendedor ù R$ 2750.

5. **Calcular a mùdia ponderada de duas provas realizadas por um aluno.**
   - Exemplo de entrada: Nota da primeira prova = 7, Peso da primeira prova = 0.4, Nota da segunda prova = 8, Peso da segunda prova = 0.6
   - Fùrmula: Mùdia ponderada = (Nota da primeira prova * Peso da primeira prova) + (Nota da segunda prova * Peso da segunda prova)
   - Exemplo de saùda: A mùdia ponderada do aluno ù 7.6.

6. **Calcular a ùrea de um terreno retangular.**
   - Exemplo de entrada: Comprimento do terreno = 10 metros, Largura do terreno = 5 metros
   - Fùrmula: ùrea = Comprimento * Largura
   - Exemplo de saùda: A ùrea do terreno ù 50 metros quadrados.

## Estruturas de Decisùo

1. **Verificar se a soma de A + B ù menor que C.**
   - Exemplo de entrada: A = 5, B = 3, C = 10
   - Exemplo de saùda: A + B ù menor que C.

2. **Ordenar trùs valores inteiros e diferentes em ordem decrescente.**
   - Exemplo de entrada: 5, 10, 3
   - Exemplo de saùda: Os nùmeros em ordem decrescente sùo 10, 5, 3.

3. **Calcular o ùndice de Massa Corporal (IMC) e determinar a condiùùo de peso de um adulto.**
   - Exemplo de entrada: Peso = 70 kg, Altura = 1.75 m
   - Fùrmula: IMC = Peso / (Altura^2)
   - Exemplo de saùda: IMC = 22.86 (Peso normal).

4. **Identificar se um nùmero ù par ou ùmpar.**
   - Exemplo de entrada: Nùmero = 7
   - Exemplo de saùda: O nùmero ù ùmpar.

5. **Realizar uma operaùùo com base em dois valores inteiros A e B.**
   - Exemplo de entrada: A = 3, B = 3
   - Exemplo de saùda: A + B = 6.
